ENDOSCAN AI LOYIHASINI RUN QILISH

1. PowerShell oching.

2. Loyiha papkasiga kiring:

   cd D:\Endos\v3

3. Birinchi marta ishlatayotgan bo'lsangiz dependency o'rnating:

   python -m pip install -r requirements.txt

4. Python dependencylar to'g'ri o'rnatilganini tekshiring:

   python -c "import fastapi, uvicorn, langchain_google_genai; print('OK')"

   Agar OK chiqsa, davom eting.

5. Java borligini tekshiring:

   javac -version
   java -version

   Agar javac va java versiya chiqarsa, Java tayyor.

6. Loyihani ishga tushiring:

   .\run_all.ps1

7. Agar PowerShell script ishlashiga ruxsat bermasa, shu komandani yozing:

   Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass

   Keyin yana ishga tushiring:

   .\run_all.ps1

8. Server ishlaganda PowerShell oynasini yopmang.

9. Brauzerda oching:

   Asosiy sayt:
   http://127.0.0.1:8000

   User login/register:
   http://127.0.0.1:8000/login
   http://127.0.0.1:8000/register

   User dashboard:
   http://127.0.0.1:8000/dashboard

   Admin login:
   http://127.0.0.1:8000/admin-login

   Admin default login:
   username: admin
   password: admin123

   Admin dashboard:
   http://127.0.0.1:8000/admin

10. Agar brauzerda ochilmasa, yangi PowerShell oynasida portlarni tekshiring:

   Get-NetTCPConnection -LocalPort 8000,8081 -State Listen

   8000 - Python FastAPI server
   8081 - Java stats backend

11. Serverni to'xtatish:

   run_all.ps1 ishlayotgan PowerShell oynasida Ctrl + C bosing.

ESLATMA:

- Java backend admin/user statistika payloadlarini beradi.
- Python loyiha login, session, asosiy API va Java bilan request/response bog'lanishini qiladi.
- UI HTML + CSS + JS orqali render bo'ladi.
